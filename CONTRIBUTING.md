# Contributing

## Кодировка
- Не меняйте кодировку файлов проекта без необходимости.
- Все файлы с русским текстом должны оставаться в `UTF-8` без BOM.
- При правках русского текста убедитесь, что редактор сохраняет файл в `UTF-8` без BOM.
- Если видите "кракозябры", не делайте дополнительные правки текста. Сначала исправьте кодировку файла.
